# O que são Heurísticas?
Heurísticas são em geral acrônimos formados pelas iniciais de uma ou mais características a serem exercitadas. A utilização da concatenação destas letras facilita a memorização (mnemônico)
Essas heurísticas podem ser usadas como checklists para guiar o profissional até seu objetivo.

# Para software com arquivos e pastas
[Centralize tudo](https://ladyogitakailas.medium.com/software-testing-heuristics-and-mnemonics-26eeb9a6f3ac#:~:text=Centralize%20Everything)<!-- {"preview":"true"} -->
- Para software em que os itens podem ser organizados em pastas: Pegue uma grande coleção de pastas com itens e mova tudo para uma única pasta ou uma única hierarquia profunda de pastas.
- Para software em que a propriedade de um item pode ser transferida entre contas: transfira a propriedade de um grande número de coisas de várias contas para uma única conta.

[Descentralize tudo](https://ladyogitakailas.medium.com/software-testing-heuristics-and-mnemonics-26eeb9a6f3ac#:~:text=Decentralize%20Everything)<!-- {"preview":"true"} -->
- Para software onde os itens podem ser organizados em pastas: Crie um grande número de pastas, cada uma com alguns itens.
- Para software que pode ser configurado para funcionar em um ambiente distribuído: Configure o sistema para que cada parte exista em sua própria máquina separada e separada das outras por sub-redes ou firewalls.

[Invertido](https://ladyogitakailas.medium.com/software-testing-heuristics-and-mnemonics-26eeb9a6f3ac#:~:text=11.-,Reverse,-Do%20things%20in)<!-- {"preview":"true"} -->
- Para software com um recurso Desfazer: execute uma ampla variedade de ações e, em seguida, desfaça todo o seu trabalho passo a passo.
- Para software com fluxo de trabalho definido: Aceite todos os valores padrão para chegar ao final do fluxo de trabalho; em seguida, use o botão Voltar para voltar a cada tela e alterar os valores.
